# SubDocTrans：基于可插拔多粒度知识增强的文档级机器翻译方法
SubDocTrans 是一个面向文档级机器翻译（Document-level Machine Translation）的模型框架，旨在通过引入多粒度知识增强机制来提升翻译质量。

相比传统句子级翻译方法，本方法能够更好地利用跨句上下文信息，从而缓解：

语义不一致
指代歧义
上下文丢失
篇章连贯性差等问题

🚀 主要特点

📄 支持文档级机器翻译建模
🔍 多粒度知识增强（句子 / 子文档 / 文档级）
🧠 上下文感知翻译机制
🔌 可插拔模块设计，易于扩展
⚡ 兼容主流 Transformer 翻译框架

🏗️ 模型结构

SubDocTrans 主要由三个核心模块组成：

子文档划分模块
将长文档划分为语义一致的子文档单元。

多粒度知识编码模块
融合句子级、子文档级和文档级上下文信息。

知识增强解码模块
在解码阶段引入多层级上下文知识，提升翻译一致性与流畅性。
